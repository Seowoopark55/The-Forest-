    에너지 믹스+🧩 재료: 치커리 + 콘 플라워 + 알로에🧪 효과: 에너지 대량 회복

    체력 믹스🧩 재료: 마리골드 + 알로에🧪 효과: 체력 회복

    체력 믹스+🧩 재료: 마리골드 + 알로에 + 콘 플라워🧪 효과: 체력 대량 회복

  • 🛡 갑옷 & 의류

    도마뱀 갑옷🧩 재료: 도마뱀 껍질🧪 효과: 기본 방어력 제공

    스텔스 갑옷🧩 재료: 도마뱀 껍질 + 나뭇잎[15]🧪 효과: 은신 능력 향상

    뼈 갑옷🧩 재료: 뼈[6] + 천[3]🧪 효과: 방어력 상승

    워름 갑옷🧩 재료: 워름 파츠[15]🧪 효과: 방어력 최고치, 돌연변이 처치 후 제작 가능

    스노우부츠🧩 재료: 토끼털[3] + 밧줄[2]🧪 효과: 눈 위에서의 이동 속도 향상

  • 🎒 가방

    가죽 물통🧩 재료: 사슴 가죽[2] + 밧줄🧪 효과: 물 저장 가능

    돌 가방🧩 재료: 멧돼지 가죽 + 천 + 밧줄[3]🧪 효과: 돌 소지량 증가

    막대기 주머니🧩 재료: 토끼털 + 밧줄[2] + 천[3]🧪 효과: 막대기 소지량 증가

    주머니🧩 재료: 토끼털[2]🧪 효과: 베리류 저장 가능

    창 가방🧩 재료: 멧돼지 가죽[2] + 사슴 가죽[2] + 천[2] + 밧줄[3]🧪 효과: 창 소지량 증가

    토끼털 부츠🧩 재료: 토끼털[3] + 밧줄[2]🧪 효과: 이동 속도 소폭 증가

    화살통🧩 재료: 토끼털[3] + 밧줄🧪 효과: 화살 최대 소지량 증가

  • ⚔️ 무기
    • 근접 무기

      강화 창🧩 재료: 창 + 천[2] + 뼈[3]🧪 효과: 공격력 증가

      강화 막대기🧩 재료: 막대기 + 천🧪 효과: 불 붙이기 가능

      강화 돌멩이🧩 재료: 돌 + 천🧪 효과: 불 붙이기 가능

      카타나🧩 재료: 협곡 동굴에서 획득🧪 효과: 빠른 공격 속도, 방어 불가

      몽둥이🧩 재료: 식인종 시체 또는 동굴🧪 효과: 공격력 높고 저지력 강함

      제작 몽둥이🧩 재료: 해골 + 막대기🧪 효과: 중간 성능, 장식 가능

      수리 도구🧩 재료: 막대기[2] + 돌 + 수액[10] + 천[2]🧪 효과: 건물 수리, 자원 수확

      테니스 라켓🧩 재료: 협곡 동굴 또는 식인종 드랍🧪 효과: 공격력 낮음, 장난감 용도

      헤어스프레이🧩 재료: 가방, 캐리어 등에서 획득🧪 효과: 화염 공격 가능, 연료 소진 시 폐기

      티미의 장난감🧩 재료: 장난감 부품[6]🧪 효과: 공격력 없음, 수집용

    • 원거리 무기

      제작 활🧩 재료: 막대기 + 천 + 밧줄🧪 효과: 기본 활, 화살 발사 가능

      현대 활🧩 재료: 협곡 동굴🧪 효과: 높은 정확도, 공격력 강력

      플린트락 권총🧩 재료: 권총 부품[8] + 폭발물로 확보🧪 효과: 공격력 높지만 재장전 느림

      조명탄 권총🧩 재료: 비행기 잔해🧪 효과: 적중 시 화염 발생

      석궁🧩 재료: 해저 동굴🧪 효과: 공격력 최상, 탄환 수거 가능

      새총🧩 재료: 막대기 + 천 + 절연 테이프🧪 효과: 작은 돌 투사

    • 투척 무기

      화염병🧩 재료: 천 + 술🧪 효과: 착탄 시 화염 발생

      다이너마이트🧩 재료: 가방 등에서 획득🧪 효과: 광역 폭발, 나무 벌목 가능

      폭발물🧩 재료: 절연테이프 + 손목시계 + 술 + 동전 + 회로판🧪 효과: 돌연변이에 강함

      점착 폭발물🧩 재료: 폭발물 + 수액🧪 효과: 표면에 달라붙음

      머리 폭탄🧩 재료: 폭발물 + 머리🧪 효과: 단일 고폭탄

      테니스공🧩 재료: 맵 전체에서 획득🧪 효과: 주의 돌리기용, 화염 가능

      작은 돌🧩 재료: 맵 전체에서 획득🧪 효과: 새총 탄환 또는 던지기

    독 무기🧩 재료: 무기 + 스노우베리 or 트윈베리[4] or 독버섯🧪 효과: 적을 중독시킴

    불 붙는 무기🧩 재료: 무기 + 천🧪 효과: 불 가능, 추위 해소

    조명 부착🧩 재료: 무기 + 손전등 + 절연 테이프🧪 효과: 야간 시야 확보
